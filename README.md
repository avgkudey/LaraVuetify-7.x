Laravel 7 Vuetify Starter Template

### Installing
* git clone https://github.com/avgkudey/LaraVuetify-Starter.git
* cd LaraVuetify
* composer install
* npm install
* php artisan serve
* npm run dev - on separate terminal
* npm run watch
